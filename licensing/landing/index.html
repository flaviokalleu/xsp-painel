Instale o painel em 3 minutos

Cole 1 comando no terminal da sua VPS. O instalador faz o resto: Docker, banco de dados, painel, certificado SSL.

1

Informe sua KEY (opcional — preenche o comando)

Se você já tem uma KEY de licença, cole abaixo para o comando vir pronto. Senão, o instalador vai pedir durante a instalação.

2

Conecte-se à sua VPS Ubuntu via SSH

Requisitos mínimos: Ubuntu 22.04+, 2 vCPU, 4 GB RAM, portas 80/443 livres, acesso root.

3

Cole este comando e pressione Enter

Tudo será instalado automaticamente. Você precisará informar seu domínio e e-mail durante o processo.

$curl -sSL https://exemplo.com/install.sh | sudo bash
O que será instalado:
  • Docker + Docker Compose
  • Painel cifrado (verificado contra adulteração)
  • Banco de dados MariaDB com senha aleatória
  • Validação online da licença a cada 5 minutos
